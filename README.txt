RAJU MASTER AI TOON MAKER

Website hii hukuruhusu kuandika script ya katuni,
kisha hutengeneza picha moja moja kwa kila mstari wa script.

Jinsi ya kutumia:
1. Fungua index.html kwa browser.
2. Andika script mfano:
   Juma alikuwa anapanda mlima mkubwa, ghafla akakutana na ndege anayeruka...
3. Bonyeza 'Tengeneza Katuni'
4. Picha zitaonekana chini ya button, kila moja ikiwa na maandishi ya script hiyo.

NB: Huu ni mfano wa offline version. Toleo la online litahitaji API kama HuggingFace au OpenAI kwa picha halisi.
